# havelange
Project Havelange - Java Spring Boot API to admin custom football tournaments.

Rest Spring Boot app

## Tech Stack

- Java 17
- Spring Boot
- Mongo DB
- Swagger2

## Requirements

For building and running the application you need:

- [Maven 3.8.6](https://maven.apache.org/download.cgi)
